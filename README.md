# Usage

<!-- usage -->

```sh-session
$ npm install -g harvest-cli
$ harvest COMMAND
running command...
$ harvest (-v|--version|version)
harvest-cli/0.1.0 linux-x64 node-v10.5.0
$ harvest --help [COMMAND]
USAGE
  $ harvest COMMAND
...
```
